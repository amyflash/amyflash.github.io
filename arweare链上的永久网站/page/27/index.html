彻底解决mysql中文乱码

改动mysql配置文件/etc/my.cnf。

[mysqld]
character-set-server=utf8 
[client]
default-character-set=utf8 
[mysql]
default-character-set=utf8

restart mysql

mysql -u root -p –default-character-set=utf8

$set names utf8
